# E-Waste Management System

A simple web-based system to manage electronic waste submissions. Users can submit e-waste items, view their submissions, and cancel if needed. Admins can view and manage all submissions.
## Features

- **User**
  - Register and login
  - Submit e-waste items (Phone, Television, Battery, Monitor, Other)
  - Track submitted items
  - Cancel submissions if needed
  - View profile

- **Admin**
  - View all submissions
  - Manage submissions (pending/collected status)
  - Access user information

- **Public**
  - View community submissions on a public page

## Tech Stack

- **Frontend:** HTML, CSS  
- **Backend:** PHP 8.x  
- **Database:** SQLite  
- **Server:** XAMPP (Apache + PHP)
